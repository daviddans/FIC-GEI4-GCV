# FIC-GEI4-GCV


Repositorio que contiene el trabajo final para la asignatura de Computación grafica y visual del grado en ingeniería informática de la facultad de informatica Da Coruña ( UDC )
Autoría: David Dans Villares
El trabajo consiste en realizar un modelado a escala del exterior de la facultad de informatica y simular la detonacion de un explosivo.
